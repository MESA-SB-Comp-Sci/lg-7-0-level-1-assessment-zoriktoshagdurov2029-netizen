## Question 1 — Describe

What is the difference between a static unit like `px` and a responsive unit like `%` or `vh`?

Static px is that it will never change, responsive like vh or % will adjust on the type of layout or size of elements based on user screen size

Describe what makes a unit responsive and why that matters when building a website.

When the unit can adjust its size to the layout differences, and does change based on screen, helpful for other layots, like if the website was made on a laptop layout, the unit can adjust to the mobile or tablet version. Helpful to not break the website for other layouts like phone or tablet.
